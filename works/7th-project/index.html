Comedy portal MVP

Refine the product strategy to build a valid MVP version of the web app and native app along with the marketing landing pages.

Product design sprint

Problem statement
Comedy goers don't have the cohesive solution to search and follow stand up comedians.
Sprint questions
  1. Is online stand-up comedian directory useful to comedy goers?
  2. Is an online venue directory useful to comedy goers?
MVP goal
  1. Build an online stand-up comedian directory
  2. Build associated features such as searching, following, and filtering

Define user flow

  • Flow by touch points and context - events / comedians / venues
  • Flow by user status - guest user vs. account user

Define JTBD + draft solution

  1. When I search for a comedy event that I want to go,
  2. I want to search by comedians I like,
  3. So I can be confident to choose the show that I would enjoy.
